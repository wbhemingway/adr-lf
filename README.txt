ADR LEAVE PARSER USER GUIDE

FIRST TIME SETUP

Extract the files. Right-click the zip file and select Extract All to a folder on your computer. Do not run the application from inside the zip folder.

Keep folders together. The bin folder must stay in the same directory as the adr_lf.exe file for the program to function.

HOW TO START

Double-click adr_lf.exe to launch the application.

To create a shortcut, right-click adr_lf.exe and select Send to then Desktop (create shortcut).

QUICK START STEPS

Set Output Folder. Click the Settings gear icon and choose where you want to save your renamed PDFs.

Load a PDF. Click Load PDF and select your document.

Review Data. The OCR engine will try to find the Name, Leave Type, and Date automatically. You can type in the boxes to fix any mistakes.

Confirm and Save. Click Confirm and Save or press Ctrl plus S to save the page and move to the next one automatically.

VIEWING TIPS

Zoom. Use your mouse wheel to zoom in and out.

Move and Pan. Click and drag on the document to move it around while zoomed in.

PRIVACY AND SECURITY

100 Percent Offline. This program does not connect to the internet. All work is done on your local computer.

No Data Storage. No personal information is saved or sent to any external server.

TROUBLESHOOTING

App won't load PDF. Make sure the bin folder was copied into the same folder as the application.

OCR is not finding text. Ensure the bin folder is present. OCR works best on clear and straight scans.
